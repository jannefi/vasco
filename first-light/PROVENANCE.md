# PROVENANCE

- NEOWISE list: authors' VASCO NEOWISE‑only catalogue (171,753 rows).
- Optical detections: SExtractor PASS2 from a pilot of <300 tiles (30′×30′ each).
- Cross‑match: nearest neighbor within 2.0″ (one‑to‑one).
- Dedupe (optical): 0.5″ cell best‑of (FLAGS→MAG_AUTO).
- This folder is **demonstration‑grade**; it is not a complete replication yet.

**Reproducibility:** see `scripts/run_first_light.sh`.
